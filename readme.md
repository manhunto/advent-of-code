# Advent of Code

This repository contains my [Advent of Code](https://adventofcode.com/) solutions in PHP.

## How to use

### Solve puzzle

```bash
php app.php app:solve [--year=2022] [--day=1] [--puzzle]
```

### List all puzzles and check results
```bash
php app.php app:list
```
